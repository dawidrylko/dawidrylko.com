# Wykres amortyzacji: Array vs Map

Wizualizacja kosztu czasowego wyszukiwania w tablicy i mapie haszującej.

## Uruchomienie

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python code.py
```

Wykres zostanie zapisany jako `amortyzacja_array_vs_map.png`.

## Parametry

W pliku `code.py` możesz zmienić:

- `n` - liczba elementów
- `k_max` - maksymalna liczba wyszukiwań na wykresie
